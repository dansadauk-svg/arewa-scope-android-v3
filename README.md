# Arewa Scope Android App v3 Stable

This is the stable APK build version for the WordPress WebView app.

It includes:
- WordPress WebView for https://arewascope.com.ng/
- Splash screen
- App icon placeholders
- Camera cutout/status bar fix
- Back button support
- File upload support
- Offline screen
- Faster WebView settings and cache behavior

Important:
This stable build temporarily removes Firebase push notification code so the APK can build cleanly first. After this version builds and installs successfully, add Firebase notifications in the next upgrade.
